THE PROJECT

About Void Matrix

A personal encryption tool built from scratch. Classical cipher architecture, terminal aesthetic, mobile-first. Runs entirely in the browser — no accounts, no servers, no tracking.

WHERE IT CAME FROM

Void Matrix started as a curiosity — what would it look like to build a functional cipher tool that felt like it belonged in a cold war signals unit, or on the screen of a hacker in an 80s film? Something that took encryption seriously but also had a distinct personality.

The cipher itself is grounded in real classical techniques — Polybius square fractionation, columnar transposition, cyclic shifting — combined into a three-key system that's simple enough to understand but strong enough to be genuinely useful for personal communication.

The aesthetic came from a love of CRT terminals, phosphor glow, the visual language of old cipher machines, and the idea that security tools don't have to look sterile and corporate to be taken seriously.

HOW IT'S BUILT

LOCAL FIRST
Everything runs in the browser. No keys, messages, or metadata are ever transmitted anywhere. The app works completely offline after the first load.
SINGLE FILE
The entire cipher engine, UI, animations, and sound design live in one HTML file. No build toolchain, no dependencies beyond two fonts and a QR library.
MOBILE FIRST
Designed primarily for iPhone. Installs as a PWA, works offline, respects safe area insets. The layout is optimized for one-handed use.
TRANSPARENT
The algorithm is fully documented and the source is open. Security comes from the keys — not from hiding how it works. Kerckhoffs's principle.

TECHNICAL SPEC

CIPHER TYPEPolybius square fractionation with columnar transposition and cyclic shifting
GRID6×6 · 36 characters · A–Z (I/J distinct) + 0–9
KEY SYSTEMThree independent keys: Grid Key, Transposition Key, Shift Sequence
PASSES1–4 iterative encryption passes
OUTPUT FORMATA–Z and 0–9 only · optional GROUP 5 chunking
SHARE FORMATVMC-MSG: base64-encoded JSON blob (keys + ciphertext + pass count)
PLATFORMSingle-file HTML/JS · no build toolchain · PWA installable
RUNTIMEBrowser-only · zero network requests during operation
SOUNDWeb Audio API · ciphertext-derived melody on encrypt
VERSIONv3.0

HONEST LIMITATIONS

Void Matrix is a classical cipher. It is not AES, RSA, or any modern cryptographic primitive. It has not been formally audited. It is not recommended for protecting sensitive data in high-threat environments.

What it is: a well-designed personal cipher that's genuinely difficult to crack without the keys, built for private communication between people who want something more interesting than sending plaintext — and who want to understand what their tools are actually doing.

Use it accordingly.
