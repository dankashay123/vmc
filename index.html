CAPABILITIES

THREE-KEY SYSTEM
Grid Key, Transposition Key, and Shift Sequence work together. Changing any one of them produces completely different ciphertext.
◈
MULTI-PASS
Run up to 4 encryption passes using the same keyset. Each pass feeds the previous ciphertext back in, dramatically increasing diffusion.
◉
SHARE BLOBS
Bundle your keys and ciphertext into a single VMC-MSG string. Your recipient pastes it and decrypts instantly — no manual key entry.
